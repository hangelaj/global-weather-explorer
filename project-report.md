# Project Report

Global Weather Explorer – Interactive Frontend Application

## Introduction

This project is an interactive web application designed to provide real-time weather information for cities around the world. The application allows users to search for a city and view current weather conditions along with a five-day forecast.

The project was developed using HTML, CSS, and JavaScript, with integration of the OpenWeather API for live data.

---

## Project Objectives

The main objectives of the project are:

* To build a fully functional interactive web application
* To allow users to search and retrieve weather data
* To display data in a clear and user-friendly interface
* To implement responsive design for different devices

---

## Features Implemented

* City search functionality
* Current weather display
* Five-day weather forecast
* Weekly calendar display
* Dynamic city image/logo
* Weather icons
* Error handling for invalid input
* Responsive design

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* OpenWeather API
* GitHub (version control and deployment)

---

## User Experience (UX)

The application is designed to be simple and intuitive.

Users can easily:

* Enter a city name
* View weather results instantly
* Navigate the interface without confusion

The layout ensures readability and accessibility across devices.

---

## Testing

Testing was carried out to ensure functionality and usability.

Tests included:

* Valid and invalid city searches
* API response handling
* Responsive design checks
* UI display validation

All major features passed testing successfully.

## Challenges Faced

* Integrating API data correctly

These challenges were resolved through testing and debugging.
